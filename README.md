
# -_18# 🎉 Event Organizer
**Система управления мероприятиями**

##  Описание проекта
Клиентское WPF-приложение для автоматизации работы организатора мероприятий. Реализована полноценная ролевая модель с разграничением прав доступа, CRUD-операции для управления событиями, модуль поиска, фильтрации и сортировки, а также административная панель для управления пользователями и просмотра базовой аналитики.

##  Реализованный функционал
✅ Авторизация и сохранение сессии  
✅ Разграничение прав: `User` | `Manager` | `Admin`  
✅ Каталог мероприятий в виде адаптивных карточек  
✅ CRUD: создание, редактирование, удаление событий  
✅ Поиск по названию/описанию  
✅ Фильтрация по статусу и типу мероприятия  
✅ Сортировка по дате, названию, бюджету  
✅ Загрузка и отображение изображений (с заглушкой)  
✅ Админ-панель: таблица пользователей, смена ролей, базовая статистика  
✅ Кнопка «Выйти» с очисткой сессии

## 🗄 Настройка базы данных
1. Убедитесь, что установлен **SQL Server** или **LocalDB**.
2. Откройте `App.config` и проверьте строку подключения:
   ```xml
   <add name="EventOrganizerDBEntities1" 
        connectionString="data source=.;initial catalog=EventOrganizerDB;integrated security=True;trustservercertificate=True;MultipleActiveResultSets=True" 
        providerName="System.Data.SqlClient" />
